# GeoMuni - Documentación del Sistema (Actualizada Mar 2026)

## Manual de Uso para Personal Municipal

### 1. Navegación y Búsqueda
*   **Buscador:** Utiliza el campo superior para localizar parcelas por su número de Padrón. El mapa se centrará automáticamente y mostrará los detalles del titular y superficie.
*   **Filtros de Capa:** Desde el panel derecho puedes encender o apagar las capas de Parcelas, Barrios e Infraestructura (Baches/Obras).

### 2. Edición de Parcelas y Barrios
*   **Creación:** Usa los botones inferiores (+ Parcela / + Barrio) para dibujar nuevas áreas. Haz clic para cada esquina y doble clic para finalizar.
*   **Modificación:** Selecciona una parcela y pulsa "Modificar Forma". Podrás arrastrar cualquier esquina para ajustar el límite exacto.
*   **Superposiciones:** El sistema no te permitirá guardar cambios si el área se cruza con otra parcela existente.

### 3. Reportes de Ciudad e Infraestructura
*   **Puntos de Incidencia:** Haz clic derecho sobre cualquier lugar para reportar un bache, luminaria rota o basural.
*   **Gestión de Cuadras:** Haz clic derecho y elige una opción de "Gestión de Cuadras" para dibujar tramos lineales de calles en obra o clausuradas.
*   **Gestión de Reportes:** Haz clic sobre cualquier bache u obra y pulsa "Gestionar Reporte" para asignar una cuadrilla responsable y actualizar el estado de la reparación.

### 4. Administración de Propietarios
*   **Base de Datos:** Accede al botón "Propietarios" para ver la lista completa de contribuyentes, registrar nuevos o editar sus datos de contacto.
*   **Vinculación:** Para asignar un dueño a una parcela, selecciónala en el mapa y utiliza el selector del panel derecho.

### 5. Seguridad y Roles
*   **Administrador:** Acceso total, incluyendo eliminación de parcelas y reportes.
*   **Editor:** Puede crear y modificar, pero no eliminar elementos críticos.
*   **Consultor:** Solo lectura. No puede realizar cambios ni reportar incidencias.
